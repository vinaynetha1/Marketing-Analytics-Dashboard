# Marketing-Analytics-Dashboard
Project Overview
Developed an end-to-end Marketing Analytics Dashboard using SQL Server, Python, and Power BI.
The dashboard provides insights into customer behavior, social media engagement, conversion performance, and customer sentiment through interactive visualizations.
